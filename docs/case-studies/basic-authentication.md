# Basic Authentication

## 1. Tinjauan Modul

**Nama Modul**  
Membuat Basic Authentication

**Deskripsi**  
Modul ini membahas cara membuat basic authentication.

**Tujuan Pembelajaran**  
Setelah menyelesaikan modul ini, peserta akan mampu membuat basic authentication.

**Ruang Lingkup Materi**  
Materi yang dibahas dalam modul ini meliputi:
- *- Data Modeling*
- *- User Interface Design*
- *- Business Logic & Procedures*
- *- Security & Authentication*
