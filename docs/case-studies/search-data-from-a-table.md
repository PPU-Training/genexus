# Search data from a table

## 1. Tinjauan Modul

**Nama Modul**  
Menampilkan Data ke Halaman Menggunakan Tabel Berdasarkan Data Pencarian

**Deskripsi**  
Modul ini akan membahas bagaimana menampilkan data dari suatu *Procedure* ke halaman menggunakan tabel berdasarkan data pencarian.

**Tujuan Pembelajaran**  
Setelah menyelesaikan modul ini, peserta diharapkan mampu menyajikan data dari sebuah *Procedure* ke dalam tampilan aplikasi dalam bentuk tabel secara terstruktur dan mudah dibaca berdasarkan data pencarian.

**Ruang Lingkup Materi**  
Materi yang dibahas dalam modul ini meliputi:
- *- Data Modeling*
- *- User Interface Design*
- *- Business Logic & Procedures*
