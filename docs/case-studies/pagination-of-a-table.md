# Pagination of a table

## 1. Tinjauan Modul

**Nama Modul**  
Menampilkan Data dalam Tabel berdasarkan Pagination

**Deskripsi**  
Modul ini membahas cara menampilkan data dalam tabel berdasarkan pagination.

**Tujuan Pembelajaran**  
Setelah menyelesaikan modul ini, peserta akan mampu menampilkan data dalam tabel berdasarkan pagination.

**Ruang Lingkup Materi**  
Materi yang dibahas dalam modul ini meliputi:
- *- Data Modeling*
- *- User Interface Design*
- *- Business Logic & Procedures*
