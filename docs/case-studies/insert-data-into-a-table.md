# Insert data into a table

## 1. Tinjauan Modul

**Nama Modul**  
Menambahkan Data ke Tabel

**Deskripsi**  
Modul ini akan membahas cara menambahkan data ke tabel.

**Tujuan Pembelajaran**  
Setelah menyelesaikan modul ini, peserta diharapkan mampu mengerti cara menambahkan data ke tabel.

**Ruang Lingkup Materi**  
Materi yang dibahas dalam modul ini meliputi:
- *- Building Web Screens & Input Forms*
- *- Business Logic & Procedures*
- *- Data Modeling*
- *- CRUD Operations (Create Operation)*
