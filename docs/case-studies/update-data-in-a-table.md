# Update data in a table

## 1. Tinjauan Modul

**Nama Modul**  
Memperbaharui Data dalam Tabel

**Deskripsi**  
Modul ini akan membahas cara memperbaharui data dalam tabel.

**Tujuan Pembelajaran**  
Setelah menyelesaikan modul ini, peserta akan mampu memperbaharui data dalam tabel.

**Ruang Lingkup Materi**  
Materi yang dibahas dalam modul ini meliputi:
- *- Building Web Screens & Input Forms*
- *- Business Logic & Procedures*
- *- Data Modeling*
- *- CRUD Operations (Update Operation)*
