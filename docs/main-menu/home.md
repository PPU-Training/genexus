# Home

Genexus adalah sebuah platform yang memungkinkan pengembang untuk membuat aplikasi web dan mobile dengan cepat dan mudah. Dengan menggunakan bahasa pemrograman yang mudah dipahami, pengembang dapat membuat aplikasi yang dapat diakses dari berbagai perangkat dan platform.

## Learning Path Syllabus
| No  | Topic                                     | Description                                                                                                                                                                                                                            | Output                                                                         |
| --- | ----------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| 1   | Overview of GeneXus 18                    | Peserta diperkenalkan GeneXus 18 sebagai platform low-code untuk pengembangan aplikasi web. Materi mencakup konsep Knowledge Base, Generator, serta arsitektur dasar GeneXus melalui penjelasan dan demo pembuatan aplikasi sederhana. | Peserta memahami konsep dasar GeneXus dan cara kerja platform secara umum.     |
| 2   | Installation & Initial Configuration      | Peserta mempelajari cara mengunduh, menginstal, dan melakukan konfigurasi awal GeneXus 18 beserta database yang digunakan agar lingkungan pengembangan siap digunakan.                                                                 | Peserta mampu melakukan instalasi dan konfigurasi awal GeneXus dengan benar.   |
| 3   | Navigating the Interface & Basic Features | Peserta mempelajari antarmuka GeneXus, struktur Knowledge Base, jenis-jenis objek, serta penggunaan tools dan properties dasar dalam pengembangan aplikasi.                                                                            | Peserta mampu memahami dan menggunakan antarmuka serta komponen utama GeneXus. |
| 4   | User Interface Design                     | Peserta mempelajari penggunaan theme, style, dan desain responsif untuk membuat tampilan aplikasi yang menarik dan mudah digunakan.                                                                                                    | Peserta mampu merancang UI aplikasi yang profesional dan responsif.            |
| 5   | Data Modeling with GeneXus                | Peserta mempelajari cara merancang struktur data menggunakan Transaction Object untuk membentuk tabel dan relasi antar data, serta menghasilkan database secara otomatis tanpa SQL manual.                                             | Peserta mampu membuat dan mengelola model data menggunakan GeneXus.            |
| 6   | Building Web Screens & Input Forms        | Peserta mempelajari cara membuat tampilan web dan form input menggunakan WebPanel untuk menampilkan dan mengelola data dalam aplikasi.                                                                                                 | Peserta mampu membuat halaman web dan form input berbasis GeneXus.             |
| 7   | Business Logic & Procedures               | Peserta mempelajari penerapan aturan bisnis dan alur proses menggunakan rules, events, dan procedure agar aplikasi berjalan sesuai kebutuhan operasional.                                                                              | Peserta mampu menerapkan logika bisnis dalam aplikasi GeneXus.                 |
| 8   | CRUD Operations                           | Peserta mempelajari cara membuat fitur Create, Read, Update, dan Delete agar aplikasi dapat mengelola data secara lengkap.                                                                                                             | Peserta mampu membangun fungsi pengelolaan data dalam aplikasi.                |
| 9   | Integrating External Resources            | Peserta mempelajari cara menghubungkan aplikasi dengan API, file, dan sistem eksternal lainnya.                                                                                                                                        | Peserta mampu mengintegrasikan GeneXus dengan sistem atau layanan eksternal.   |
| 10  | Security & Authentication                 | Peserta mempelajari penerapan sistem login, manajemen user, role, dan hak akses untuk melindungi data dan fitur aplikasi.                                                                                                              | Peserta mampu menerapkan keamanan dan autentikasi dalam aplikasi.              |
| 11  | Testing & Debugging                       | Peserta mempelajari cara melakukan testing, menggunakan debugger, membaca log, dan menangani error.                                                                                                                                    | Peserta mampu memastikan aplikasi berjalan stabil dan bebas dari kesalahan.    |
| 12  | Development Team Management               | Peserta mempelajari pengembangan aplikasi secara tim, termasuk version control, sinkronisasi perubahan, dan pengelolaan versi aplikasi.                                                                                                | Peserta mampu bekerja secara kolaboratif dalam pengembangan aplikasi GeneXus.  |
